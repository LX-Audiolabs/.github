# LX Audiolabs

**Open-source CLAP audio plugins** — built with [truce](https://truce.audio/) and Rust.

---

### 🎛 Plugins

| Plugin | Type | Role |
|--------|------|------|
| **[Equilibrium](https://github.com/LX-Audiolabs/lx-audiolabs)** | 5-Band Spectral Balancer | Master Bus — precision band correction |
| **[Meridian](https://github.com/LX-Audiolabs/lx-audiolabs)** | Group Track Sculptor | Tracks & Buses — EQ, compressor, saturation |
| **[Aether](https://github.com/LX-Audiolabs/lx-audiolabs)** | Headphone Correction | Monitoring FX — Harman target curve + crossover |
